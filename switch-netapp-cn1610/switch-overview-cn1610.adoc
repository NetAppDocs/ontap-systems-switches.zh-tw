= NetApp CN1610交換器的安裝與設定概述
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
CN1610 是一款高頻寬、可管理的二層交換機，提供 16 個 10 千兆小型可插拔增強型 (SFP+) 連接埠。

此交換器包含冗餘電源和風扇托架，支援熱插拔，實現高可用性。這款 1U 交換器可安裝在標準的 19 吋NetApp 42U 系統機櫃或第三方機櫃中。

此交換器支援透過控制台連接埠進行本機管理，或透過網路連線使用 Telnet 或 SSH 進行遠端管理。 CN1610 包含一個專用的 1 千兆乙太網路 RJ45 管理端口，用於帶外交換器管理。您可以透過在命令列介面 (CLI) 中輸入命令或使用基於 SNMP 的網路管理系統 (NMS) 來管理交換器。
