:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
對於NVIDIA SN2100 交換器的安裝和維護，請務必查看機櫃和導軌套件的組件和零件號碼清單。

您可以使用交換器隨附的標準支架將NVIDIA SN2100 交換器 (X190006) 安裝到NVIDIA雙/單交換器機櫃。
