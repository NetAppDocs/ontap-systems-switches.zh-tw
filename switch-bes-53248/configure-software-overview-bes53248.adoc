= BES-53248交換器的軟體安裝工作流程
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
若要安裝和設定 BES-53248 叢集交換器的軟體，請依照下列步驟操作：

.image:https://raw.githubusercontent.com/NetAppDocs/common/main/media/number-1.png["一"]link:configure-install-initial.html["配置交換機"]
[role="quick-margin-para"]
配置 BES-53248 叢集交換器。

.image:https://raw.githubusercontent.com/NetAppDocs/common/main/media/number-2.png["二"]link:configure-efos-software.html["安裝 EFOS 軟體"]
[role="quick-margin-para"]
在 BES-53248 叢集交換器上下載並安裝乙太網路結構作業系統 (EFOS) 軟體。

.image:https://raw.githubusercontent.com/NetAppDocs/common/main/media/number-3.png["三"]link:configure-licenses.html["安裝 BES-53248 叢集交換器的許可證"]
[role="quick-margin-para"]
（可選）透過購買和安裝更多許可證來新增連接埠。交換器基本型號授權配備 16 個 10GbE 或 25GbE 連接埠和 2 個 100GbE 連接埠。

.image:https://raw.githubusercontent.com/NetAppDocs/common/main/media/number-4.png["四"]link:configure-install-rcf.html["安裝參考設定檔（RCF）"]
[role="quick-margin-para"]
在 BES-53248 叢集交換器上安裝或升級 RCF，然後在應用 RCF 後驗證連接埠是否具有額外的授權。

.image:https://raw.githubusercontent.com/NetAppDocs/common/main/media/number-5.png["五"]link:configure-ssh.html["在 BES-53248 叢集交換器上啟用 SSH"]
[role="quick-margin-para"]
如果您使用乙太網路交換器健康監視器 (CSHM) 和日誌收集功能，請在交換器上啟用 SSH。

.image:https://raw.githubusercontent.com/NetAppDocs/common/main/media/number-6.png["六"]link:reset-switch-bes-53248.html["將交換器重設為出廠預設值"]
[role="quick-margin-para"]
清除 BES-53248 群集交換器設定。
