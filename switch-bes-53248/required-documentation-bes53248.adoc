== 博通文檔

要設定 BES-53248 叢集交換機，您需要從博通支援網站取得以下文件： https://www.broadcom.com/support/bes-switch["博通乙太網路交換器產品線"^]

[cols="1,2"]
|===
| 文件標題 | 描述 


 a|

 a| 
提供在典型網路中使用 BES-53248 交換器的範例。



 a| 
_EFOS CLI 指令參考 v3.4.3_
 a| 
描述用於檢視和設定 BES-53248 軟體的命令列介面 (CLI) 命令。



 a| 
EFOS 入門指南 v3.4.3
 a| 
提供有關 BES-53248 交換器的詳細資訊。



 a| 
EFOS SNMP 參考指南 v3.4.3
 a| 
提供在典型網路中使用 BES-53248 交換器的範例。



 a| 
_EFOS 縮放參數和值 v3.4.3_
 a| 
描述 EFOS 軟體在支援的平台上交付和驗證時所採用的預設擴充參數。



 a| 
EFOS 功能規格 v3.4.3
 a| 
描述 EFOS 軟體在支援平台上的規格。



 a| 
EFOS 版本說明 v3.4.3
 a|

 a|

 a|

|===
