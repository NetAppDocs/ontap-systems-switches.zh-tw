= Cisco Nexus 9364D-GX2A 交換器的連接埠詳細信息
:allow-uri-read: 


[role="lead"]
Cisco Nexus 9364D-GX2A 交換器是Cisco Nexus 9000 平台的一部分，可以安裝在NetApp系統機櫃中。  9364D-GX2A 交換器可讓您建置具有兩個以上節點的ONTAP叢集。

|===

 a| 
乙太網路1/63、乙太網路1/64

400GbE ISL
