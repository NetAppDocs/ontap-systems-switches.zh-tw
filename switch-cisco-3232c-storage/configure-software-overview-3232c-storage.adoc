= Cisco Nexus 3232C 儲存交換器的軟體安裝工作流程
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
若要安裝和設定Cisco Nexus 3232C 交換器的軟體並安裝或升級參考設定檔 (RCF)，請依照下列步驟操作：

.image:https://raw.githubusercontent.com/NetAppDocs/common/main/media/number-1.png["一"]link:setup-switch-3232c-storage.html["配置交換機"]
[role="quick-margin-para"]
配置 3232C 儲存交換器。

.image:https://raw.githubusercontent.com/NetAppDocs/common/main/media/number-2.png["二"]link:prepare-install-cisco-nexus-3232c-storage.html["準備安裝 NX-OS 軟體和 RCF"]
[role="quick-margin-para"]
Cisco NX-OS 軟體和參考組態檔 (RCF) 必須安裝在 Cisco 3232C 儲存交換器上。

.image:https://raw.githubusercontent.com/NetAppDocs/common/main/media/number-3.png["三"]link:install-nx-os-software-3232c-storage.html["安裝或升級 NX-OS 軟體"]
[role="quick-margin-para"]
在 Cisco 3232C 儲存交換器上下載並安裝或升級 NX-OS 軟體。

.image:https://raw.githubusercontent.com/NetAppDocs/common/main/media/number-4.png["四"]link:install-rcf-software-3232c-storage.html["安裝 RCF"]
[role="quick-margin-para"]
首次設定 Cisco 3232C 儲存交換器後，安裝 RCF。

.image:https://raw.githubusercontent.com/NetAppDocs/common/main/media/number-5.png["五"]link:upgrade-rcf-software-3232c-storage.html["升級 RCF"]
[role="quick-margin-para"]
根據需要升級您現有的 RCF 版本。

.image:https://raw.githubusercontent.com/NetAppDocs/common/main/media/number-6.png["六"]link:configure-ssh-keys.html["驗證 SSH 配置"]
[role="quick-margin-para"]
驗證交換器上是否啟用了 SSH 以使用乙太網路交換器健康監視器 (CSHM) 和日誌收集功能。

.image:https://raw.githubusercontent.com/NetAppDocs/common/main/media/number-7.png["七"]link:reset-switch-3232c-storage.html["將交換器重設為出廠預設值"]
[role="quick-margin-para"]
清除 3232C 儲存交換器設定。
