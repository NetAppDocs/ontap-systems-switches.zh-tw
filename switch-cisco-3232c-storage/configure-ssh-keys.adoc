= 請檢查您的 SSH 配置
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
如果您正在使用乙太網路交換器健康監視器 (CSHM) 和日誌收集功能，請驗證交換器上是否啟用了 SSH 和 SSH 金鑰。

.步驟
. 確認 SSH 已啟用：
+
[listing, subs="+quotes"]
----
(switch) *show ssh server*
ssh version 2 is enabled
----
. 請確認 SSH 金鑰已啟用：
+
`show ssh key`

+
