對於Cisco Nexus 3232C 交換器的安裝和維護，請務必查看設定和網路需求。



== 配置要求

您需要為交換器準備適當數量和類型的纜線及纜線連接器。根據您初始設定的交換器類型，您需要使用隨附的主控台纜線連接到交換器主控台連接埠；此外，您還需要提供特定的網路資訊。



== 網路需求

所有交換器配置都需要以下網路資訊：

* 管理網路流量的 IP 子網
* 每個儲存系統控制器和所有適用交換器的主機名稱和 IP 位址
* 大多數儲存系統控制器透過連接到乙太網路服務連接埠（扳手圖示）透過 e0M 介面進行管理。在AFF A800和AFF A700系統中，e0M 介面使用專用乙太網路連接埠。
