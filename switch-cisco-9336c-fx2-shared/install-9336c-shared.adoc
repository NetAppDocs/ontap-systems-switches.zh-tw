請依照下列指示設定Cisco Nexus 9336C-FX2共用交換器。

* 必要的共享交換器文件、控制器文件和ONTAP 參考文件。請參閱 link:required-documentation-9336c-shared.html["Cisco Nexus 9336C-FX2共享交換器的文件要求"] 和 https://["NetApp ONTAP 產品文件"^]。
