= 安裝Cisco Nexus 9336C-FX2 共用交換機
:hardbreaks:
:allow-uri-read: 
:nofooter: 
:icons: font
:linkattrs: 
:imagesdir: ../media/


[role="lead"]
請依照以下說明設定Cisco Nexus 9336C-FX2 共用交換器。

.開始之前
請確保您擁有以下物品：

* 需要共用交換器文件、控制器文件和ONTAP文件。看link:required-documentation-9336c-shared.html["Cisco Nexus 9336C-FX2 共享交換器的文件需求"]和 https://docs.netapp.com/us-en/ontap/index.html["NetApp ONTAP 文檔"^]。
* 適用的許可證、網路和設定資訊以及線纜。
* 已完成佈線工作表。看link:cable-9336c-shared.html["完成Cisco Nexus 9336C-FX2 佈線工作表"]。有關佈線的更多信息，請參閱 https://hwu.netapp.com["Hardware Universe"]。


.步驟
. 將交換器、控制器和 NS224 NVMe 儲存架安裝到機架上。
+
參見link:../switch-cisco-9336c-fx2/install-switch-and-passthrough-panel-9336c-cluster.html["貨架安裝說明"]學習如何將交換器安裝到NetApp機櫃。

. 開啟交換器、控制器和 NS224 NVMe 儲存架的電源。


.下一步是什麼？
（可選） link:install-switch-and-passthrough-panel-9336c-shared.html["在NetApp機櫃中安裝Cisco Nexus 9336C-FX2 交換機"]。否則，請前往 link:setup-and-configure-9336c-shared.html["配置交換機"]。
