= 驗證SSH組態
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
如果您使用的是乙太網路交換器健全狀況監視器（ CSHM ）和記錄收集功能、請確認叢集交換器上已啟用 SSH 和 SSH 金鑰。

.步驟
. 確認SSH已啟用：
+
[listing, subs="+quotes"]
----
(switch) *show ssh server*
ssh version 2 is enabled
----
. 確認 SSH 金鑰已啟用：
+
`show ssh key`

+
