= 完整的Cisco Nexus 3132Q-V 佈線工作表
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
如果您想記錄支援的平台，請下載此頁面的 PDF 檔案並填寫佈線工作表。

範例佈線工作表提供了從交換器到控制器的建議連接埠分配範例。空白工作表提供了一個模板，您可以在設定叢集時使用該模板。

每個交換器可以配置為一個 40GbE 連接埠或 4 個 10GbE 連接埠。
